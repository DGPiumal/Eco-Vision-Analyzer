# 🔬 Eco-Vision Analyzer

An advanced, user-friendly image processing application built with **Streamlit** and **OpenCV**. Designed for scientific research, this tool automates the detection, segmentation, and statistical analysis of micro-particles (such as microplastics). 

### Key Features:
* **Multiple Pipelines:** Supports Nile Red (fluorescence) and Optical (FFT + Gradient) image processing.
* **Interactive ROI:** Click-to-center region of interest adjustment for precise analysis.
* **Automated Data Reporting:** Generates statistical distributions and exportable CSV reports.
* **User-Friendly UI:** Clean, intuitive dashboard built purely in Python.
